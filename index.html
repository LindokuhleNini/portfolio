DVD Rental Application

Java | SQL | Derby Database

This is a client-server application that I created as part of a project assigned to me during my second year of university. The goal was to automate all of the daily operations of a small DVD Rental Store in order to make management easier. The client side of the application is in charge of receiving user input, while the server side is in charge of interacting with the database. The most difficult aspect of this project was grasping the concept of multi-threading. However, after hours of research and frustrating errors, I finally understood how it works and passed with a distinction.

Vaccine Registration Portal Clone

HTML/CSS | Javascript | ExpressJS | NodeJS | MongoDB | Twilio API

During my internship last year, I was tasked with creating a clone of the vaccination registration portal. This project's goal was to introduce us to MongoDB, NodeJS, and the Express framework. The user fills out a few forms, just like the real Vaccination Registration Portal, and then the data is saved to a database. The difference is that I also had to include a Whatsapp chatting bot for users who prefer to register via Whatsapp. So I had to learn how to work with APIs as well.
